\\
\textbf{Solution: }

\begin{align}
\myvec{
   \vec{B}-\vec{A} & \vec{C}-\vec{A}
 }^T = \myvec{
   0 & -12 \\
   0 & -4
   }\\
 \xleftrightarrow[]{R_2 \rightarrow {R_2-\frac{1}{3}R_1}}
 \myvec{
   0 & -12
   \\
   0 & 0
   }
\end{align}
Since the rank of the Collinearity matrix is $1$, the points are collinear
